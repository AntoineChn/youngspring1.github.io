MIT:The Analytics Edge 笔记10－数据收集

汇总了一下，MIT课程 15.071x The Analytics Edge 里面，收集到的数据，以及它们的来源。
你也可以在 这个页面 下载到所有跟课程相关的CSV数据、课件、R脚本。

Unit1

  • WHO的世界健康数据 WHO.csv
    来自 Global Health Observatory Data Repository
    可以按照主题、分类、指标、国家来获取。

  • USDA的食物数据 USDA.csv
    数据来自 USDA National Nutrient Database for Standard Reference

  • 芝加哥的犯罪数据 mvtWeek1.csv
    数据由FBI统计，由cityofchicago公开

  • New York Stock Exchange (NYSE)的股价数据
    IBM历史股价 IBMStock.csv
    通用电气历史股价 GEStock.csv
    宝洁历史股价 ProcterGambleStock.csv
    可口可乐历史股价 CocaColaStock.csv
    波恩历史股价 BoeingStock.csv
    数据来自 infochimps

  • 美国人口普查数据 CPSData.csv
    数据来自 Current Population Survey (CPS)
    另付CPSData里面的地区代码 MetroAreaCodes.csv 和 CountryCodes.csv

Unit2

  • 影响酒价格的因素
    wine.csv 和 wine_test.csv
    来自研究论文 Liquid Assets

  • 棒球比赛数据 baseball.csv
    来自 baseball-reference

  • 篮球比赛数据
    NBA_train.csv 和 NBA_test.csv
    来自 basketball-reference

  • 1983-2008全球气候变化状况 climate_change.csv
    其中，温度(Temp)数据来自 Climatic Research Unit at the University of East Anglia
    大气成分(CO2, N2O, CH4, CFC.11, CFC.12)数据来自 ESRL/NOAA Global Monitoring Division
    颗粒物(Aerosols)数据来自 Godard Institute for Space Studies at NASA
    TSI(total solar irradiance)数据来自 SOLARIS-HEPPA project website
    multivariate El Nino Southern Oscillation index (MEI)数据来自 ESRL/NOAA Physical Sciences Division

  • The Programme for International Student Assessment (PISA)国际留学生评价程序
    pisa2009train.csv 和 pisa2009test.csv
    这些数据来自美国国家教育统计中心的文件 2009 PISA Public-Use Data Files。
    注意，使用这些数据的时候，你要遵守 NCES data use agreement。

  • 流感趋势数据 FluTrain.csv
    我们可以使用 Google Trends 来观察人们都在搜索什么内容。如果搜索流感信息的人很多，那么可能就要爆发流感啦！是否真的爆发流感呢，U.S. Centers for Disease Control and Prevention会公开influenza-like illness (ILI)这样的信息。

  • 美国各州信息
    statedata.csv

  • 现代汽车 Hyundai Elantra 在美国的销售状况
    elantra.csv

Unit3

  • 病人信息 framingham.csv
    数据来自这项研究 BioLINCC

  • 总统大选数据 PollingData.csv
    上面的CSV可能有些问题，你也许想使用这份处理过后的 PollingData_Imputed.csv
    数据来自 RealClearPolitics

  • 流行歌曲数据 songs.csv
    数据来自 Wikipedia, Billboard.com, 和 EchoNest。

  • 罪犯假释数据 parole.csv
    数据来自 United States 2004 National Corrections Reporting Program

  • 借款人信用数据 loans.csv
    数据来自 LendingClub

Unit4

  • 美国最高法院斯蒂文森大法官判例数据 stevens.csv
    数据来自 Supreme Court Forecasting Project

  • 病人信息 ClaimsData.csv.zip（这个有点大，解压后17M，慎重下载）
    数据来自 DE-SynPUF dataset

  • 波士顿房价数据 boston.csv
    数据来自 UCI Machine Learning Repository

  • 投票动机数据 gerber.csv
    数据来自研究项目 2008 research paper

  • 字母识别数据 letters_ABPR.csv
    数据来自 UCI Machine Learning Repository

  • 人口普查数据 census.csv
    数据来自 UCI Machine Learning Repository

Unit5

  • 邮件数据 energy_bids.csv
    数据来自 TREC Legal Track

  • wiki页面Language的编辑日志 wiki.csv

  • 医院处方数据 clinical_trial.csv
    数据来自 Pubmed

  • 垃圾邮件数据 emails.csv

Unit6

  • 电影信息页面
    你需要自己保存和解析。

  • 单词出现频率数据 dailykos.csv
    数据来自 Daily Kos

  • 航空旅客里程数据 AirlinesCluster.csv
    数据来自书籍 Data Mining for Business Intelligence

  • 股票涨跌数据 StocksCluster.csv
    数据来自 infochimps

Unit7

  • 罪犯地点数据 mvt.csv
    数据来自 芝加哥警察局

  • 谋杀案件数据 murders.csv
    数据由FBI统计，公开于WIKI

  • MIT留学生信息 intlall.csv
    数据来自 MIT International Students Office
